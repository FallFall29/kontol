Overview

API ini digunakan untuk membuat QRIS dinamis, mengecek status pembayaran, melihat riwayat transaksi, dan menerima callback dari SMP Payment. Semua endpoint menggunakan format JSON.

PropertyValue
Base URLhttps://apinyapadam.vercel.app
Auth (Public API)?api_key=xxx — URL parameter
Auth (Callback)/:key — URL path
Minimal DepositRp500
Timeout Transaksi5 menit (300 detik)
StorageUpstash Redis
Autentikasi

Semua endpoint Public API menggunakan API Key sebagai URL parameter.

Format
GET/POST https://apinyapadam.vercel.app/api/qris/[endpoint]?api_key=YOUR_API_KEY

Response jika API Key tidak valid:

{ "success": false, "message": "API key tidak valid" }
Buat QRIS
POST /api/qris/create?api_key=xxx

Membuat QRIS dinamis baru. Mengembalikan gambar QR base64 dan ID transaksi.

Request Body (JSON)
FieldTypeRequiredKeterangan
amountintegerWAJIBNominal dalam Rupiah, minimal 500
Contoh Request
curl -X POST \
  "https://apinyapadam.vercel.app/api/qris/create?api_key=fallshop-api-key" \
  -H "Content-Type: application/json" \
  -d '{"amount": 10000}'
Response Sukses (200)
{
  "success": true,
  "data": {
    "id": "1773462159100723",
    "amount": 10000,
    "qris_string": "00020101021126670016...",
    "qr_image": "data:image/png;base64,iVBORw0KGgo...",
    "expired_in": 300
  }
}
Response Error (400)
{ "success": false, "message": "Minimal deposit Rp500" }
💡 id — simpan untuk cek status pembayaran.
💡 qr_image — PNG base64, langsung tampilkan di HTML atau kirim ke Telegram.
💡 qris_string — string QRIS mentah jika ingin generate QR sendiri.

Riwayat Transaksi
GET /api/qris/history?api_key=xxx

Mengambil daftar transaksi terbaru. Mendukung pagination dan filter status.

Query Parameters
ParameterTypeRequiredKeterangan
api_keystringWAJIBAPI Key autentikasi
limitintegerOPSIONALJumlah data (default: 20, max: 100)
pageintegerOPSIONALNomor halaman (default: 1)
statusstringOPSIONALFilter: SUCCESS / PENDING / EXPIRED
Contoh Request
# 10 terbaru

# Filter sukses saja

Callback SMP Payment

SMP Payment mengirim HTTP POST ke URL callback setiap transaksi berhasil.

URL Callback
https://apinyapadam.vercel.app/api/callbackqris/{KEY}
⚠️ Ganti {KEY} dengan nilai CALLBACK_SECRET di Vercel. Contoh: fallshop
https://apinyapadam.vercel.app/api/callbackqris/fallshop
Payload yang Diterima
{
  "us_username": "fallshop",
  "tr_id": "018230241703",
  "issuer": "DANA",
  "PayerName": "Jhon Doe",
  "rrn": "018230241703",
  "amount": { "value": "10000.00" },
  "saldo_akhir": 62000,
  "timestamp": "2026-03-14 11:31:52"

Python

Install: pip install requests

# bot.py / termux
import requests, base64

BASE_URL = "https://apinyapadam.vercel.app"
API_KEY  = "fallshop-api-key"

# Buat QRIS
res = requests.post(
    f"{BASE_URL}/api/qris/create",
    params={"api_key": API_KEY},
    json={"amount": 10000}
)
trx = res.json()["data"]
print("ID:", trx["id"])

# Simpan gambar QR
b64 = trx["qr_image"].split(",")[1]
with open("qris.png", "wb") as f:
    f.write(base64.b64decode(b64))

# Cek status
res = requests.get(
    f"{BASE_URL}/api/qris/status",
    params={"api_key": API_KEY, "id": trx["id"]}
)
print("Status:", res.json()["data"]["status"])

# Riwayat
res = requests.get(
    f"{BASE_URL}/api/qris/history",
    params={"api_key": API_KEY, "limit": 10, "status": "SUCCESS"}
)
for item in res.json()["data"]:
    print(f"Rp{item['amount']} - {item['status']}")
JavaScript
const BASE_URL = "https://apinyapadam.vercel.app";
const API_KEY  = "fallshop-api-key";

// Buat QRIS
const res = await fetch(`${BASE_URL}/api/qris/create?api_key=${API_KEY}`, {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ amount: 10000 })
});
const { data } = await res.json();
console.log("ID:", data.id);

// Cek status
const r = await fetch(`${BASE_URL}/api/qris/status?api_key=${API_KEY}&id=${data.id}`);
const result = await r.json();
console.log("Status:", result.data.status);

// Polling otomatis tiap 5 detik
function pollStatus(id, onSuccess) {
  const t = setInterval(async () => {
    const r = await fetch(`${BASE_URL}/api/qris/status?api_key=${API_KEY}&id=${id}`);
    const d = await r.json();
    if (d.data.status !== "PENDING") {
      clearInterval(t);
      onSuccess(d.data);
    }
  }, 5000);
}
PHP
<?php
$BASE_URL = "https://apinyapadam.vercel.app";
$API_KEY  = "fallshop-api-key";

// Buat QRIS
$ch = curl_init();
curl_setopt_array($ch, [
  CURLOPT_URL => "$BASE_URL/api/qris/create?api_key=$API_KEY",
  CURLOPT_POST => true,
  CURLOPT_POSTFIELDS => json_encode(["amount" => 10000]),
  CURLOPT_HTTPHEADER => ["Content-Type: application/json"],
  CURLOPT_RETURNTRANSFER => true
]);
$res   = json_decode(curl_exec($ch), true);
$id    = $res["data"]["id"];
$qr_b64 = explode(",", $res["data"]["qr_image"])[1];
file_put_contents("qris.png", base64_decode($qr_b64));

// Cek status
curl_setopt($ch, CURLOPT_URL, "$BASE_URL/api/qris/status?api_key=$API_KEY&id=$id");
curl_setopt($ch, CURLOPT_POST, false);
curl_setopt($ch, CURLOPT_HTTPGET, true);
$status = json_decode(curl_exec($ch), true);
echo $status["data"]["status"];
?>
